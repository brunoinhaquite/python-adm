# python-adm-focused-training
# Lógica de Programação e Manipulação de Dados para Administração 📊🐍

Este repositório contém os exercícios e exemplos práticos da **Aula 04**, ministrada pelo **Prof. Sérgio Assunção Monteiro, DSc**. O objetivo central é aplicar conceitos de lógica de programação e análise de dados em cenários reais do cotidiano administrativo e de gestão.

## 🎯 Objetivos do Projeto
- Praticar o uso de estruturas condicionais para tomada de decisão.
- Implementar loops (comandos iterativos) para processamento de listas de dados.
- Utilizar a biblioteca **Pandas** para leitura, filtragem e manipulação de arquivos Excel.
- Gerar visualizações de dados estatísticos com **Seaborn** e **Matplotlib**.

## 📑 Conteúdo do Notebook
O projeto está dividido em 5 exercícios principais:

1.  **Lógica e Condicionais:** Classificação de desempenho de vendas e análise de margem de lucro.
2.  **Comandos Iterativos:** Cálculo de médias de vendas semanais e contagem de estoque/devoluções.
3.  **Pandas e Arquivos:** Importação de dados de arquivos `.xlsx` e aplicação de filtros operacionais.
4.  **Manipulação de Dados:** Criação de novas métricas como "Receita Total" e "Ticket Médio".
5.  **Visualização de Dados:** Geração de gráficos de barras (por produto) e de linhas (evolução temporal).

## 🛠️ Tecnologias Utilizadas
- **Python 3**
- **Pandas:** Para manipulação de DataFrames.
- **Seaborn & Matplotlib:** Para criação de gráficos estatísticos.
- **Jupyter Notebook / Google Colab**
